# Spellid

Adds Spell ID and Icon ID to tooltips for spells and items.

## Information

- **Version:** 1.2
- **Author:** enjoymygripz
- **CurseForge:** https://www.curseforge.com/wow/addons/spellid

## Installation

1. Download the latest release
2. Extract to `World of Warcraft\Interface\AddOns\`
3. Restart WoW
